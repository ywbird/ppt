
# Procedural Animation

## Introduction

 - 게임 해본적?

 - ex) minecraft, tetris, hollow knight, celeste
 
 - what if 플레이어가 하는 행동에 동적으로 반응하게 want?

 - ex) Rain World, Critter Crosser
 
## Procedural Animation?
 
 - 수학적 코드로 생성되는 애니메이션
 - $(r\cos\theta, r\sin\theta)$, $Am+B(1-m)$, Bezier Curve

 - Keyframe 비교
 - PA : 알고리즘을 통해 생성 실시간 생성 -> 동적
 - KeyFrame : 미리 생성 -> 동적 X
